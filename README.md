# 두 왕국의 전쟁

중세에서 현대까지, 두 왕국이 전쟁하는 실시간 시뮬레이션 게임.
단일 HTML 파일로 구성된 브라우저 게임입니다.

## 플레이

[GitHub Pages에서 바로 플레이](https://huisu-hwang.github.io/two-kingdoms-war/)

## 주요 기능

- **3가지 시대 모드**: 중세 / 발전형 / 현대
- **9종 병종**: 보병, 궁수, 기병, 창병, 석궁병, 공성병기, 소총병, 탱크, 전투기
- **병종 상성**: 가위바위보식 상성 시스템
- **무기 연구**: 8종의 연구로 군대 강화 및 신병종 잠금 해제
- **3가지 진영 모드**: 청룡 통치 / 주작 통치 / 2인 모드 / 관전자
- **도시 점령 & 주둔**: 13개 도시를 두고 전략적 점령전
- **방어 보너스**: 자기 영토에서 1.35배 효율
- **평화 / 전쟁 사이클**: 자동 휴전과 선전포고

## 조작법

- 좌클릭 드래그: 부대 선택
- 우클릭(빈 곳/적 도시): 이동/공격 명령
- 우클릭(아군 도시): 주둔
- Space: 일시정지
- 1/2/4: 속도 조절
- TAB(2인 모드): 진영 전환
- ESC: 선택 해제

## 승리 조건

적국의 인구가 0이 되거나 적국의 모든 도시를 함락하면 승리.
